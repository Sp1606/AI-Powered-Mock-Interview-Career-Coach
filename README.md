# IntervAI - AI-Powered Mock Interview & Career Coach 🤖🚀

IntervAI is a cutting-edge, recruiter-grade web application designed to help developers practice coding, behavioral, and architectural mock interviews. Tailored specifically for branches like **Mobile/Flutter Development**, **Python/ML Engineering**, **Full-Stack Development**, and **Data Analytics**, it prepares developers to ace their interviews and land jobs.

---

## 🌟 Key Features

### 1. Multi-Role Mock Interviews
*   **Flutter Developer**: Custom questions covering Dart list structures, mobile state management (BLoC, Riverpod), widget rebuilding, and performance profiling.
*   **Python / ML Engineer**: Algorithmic math functions using vectorized NumPy arrays, cross-validation, feature standardization, and Principal Component Analysis (PCA).
*   **Full-Stack Engineer**: Javascript rate-limiter implementations, SQL/NoSQL schema tradeoffs, Redis caching, and Express API optimizations.
*   **Data Analyst**: SQL window functions (`DENSE_RANK`), outlier detection, and data imputation algorithms.

### 2. Dual Simulation Modes
*   **Live AI Mode (Google Gemini)**: Securely input your Gemini API Key in the Settings page to experience live, custom interviews where a generative AI agent analyzes your resume, conducts follow-up questions, and dynamically evaluates your code answers.
*   **Offline Simulator Mode (Fallback)**: Don't have an API key? Practice instantly with a rich, built-in database of realistic coding challenges and behavioral reviews that run 100% offline.

### 3. Split-Pane Code Terminal & Console
*   Write solutions directly in the code editor (supporting syntax tab shifts and auto-generated line counts).
*   Run code with a simulated browser sandbox that executes validation logic and parses console logs.
*   Request immediate hints directly from the mock interviewer.

### 4. Recruiter-Grade Scorecard & Report
*   Overall score match rendered with SVG progress indicators.
*   Competency breakdown bars evaluating **Logic & Correctness**, **Problem Solving Approach**, **System Architecture**, and **Communication**.
*   Detailed review for each question highlighting strengths, suggestions, and recommended answers.

### 5. Web Speech Voice Integrations
*   **Text-to-Speech (TTS)**: Let the AI interviewer read questions out loud.
*   **Speech-to-Text (STT)**: Practice answering behavioral questions vocally with inline transcription.

---

## 🛠️ Technology Stack
*   **Core**: HTML5, Vanilla JavaScript.
*   **Styling**: Modern CSS3 (Dark-mode first, glassmorphism, responsive grids, custom CSS variables, and keyframe animations).
*   **APIs**: Google Gemini Developer API (Generative Language API), Web Speech API.

---

## 🚀 Getting Started

No installation, compilation, or web servers are required!

1.  Clone this repository or download the source code.
2.  Locate `index.html` and double-click it (or drag and drop it into Google Chrome, Safari, or Firefox).
3.  Choose your target role and difficulty, and begin practicing!
4.  *(Optional)* Go to the **API Settings** tab to save your Google Gemini API key for live AI evaluations.
